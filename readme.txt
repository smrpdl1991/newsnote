== Description ==
NewsNote is a clean, elegant and stylish WordPress multipurpose theme, designed for a web news portal, magazine and blog. It is fully widget based site. You can implement different layouts from the widgets and customizer options.The theme has optimized codes and awesome supports. You can create news journals, modern fashion magazine, travel blogging & magazine, clean and minimal news sites, blogging site and even more.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Go to Dashboard > Appearance > Customize and change settings
5. Go to Dashboard > Appearance > Widgets and change settings
